Ionic E-Commerce starter
==========================================
This is an E-commerce starter theme, which you can use for building an e-commerce mobile application with Ionic framework.


This starter has the following features.
------------------------------------------
- Loading indicators at products page
- Pull to refresh at products page
- List view in products page
- Side by side view in products page
- Add address, card, coupon pages
- Search page
- Add a write new review page
- Form Validations
- Supports embedded videos
- Filter page functionality
and lot more.


Plugins integrated
------------------------------------------
- Social Sharing plugin integrated
- InAppBrowser Plugin integration
- ionic-ratings plugin integrated
